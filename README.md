# Description:this is a coded choice game that varies from what choices you make early in the story. The variety of different paths keeps the user engaged without any over the top story lines. it uses many If snf Elif statments to execute the game and writtten in python3. 
## requirments
              * python3
              *if/elif/else functions
              *decision tree (seperate from code)
## Insatalation:
                *open repository in github
                *clone repository and push to visual studio
                *if you are using visual studio, which is what the game was coded on, in order to run the game you will need to press the grey play button located in the top right corner
                * if inputed correctly it should greet the player with text and two choices that will start the story and the game.
